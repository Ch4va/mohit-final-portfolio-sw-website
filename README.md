# Mohit — Digital Creator Portfolio Website(SOFTWARE)

A premium, interactive, and modern digital portfolio website built with rich animations, custom interactions, and clean layouts.

## 🚀 Live Demo & Repository
- **Repository Name on GitHub**: `mohit-final-portfolio-sw-website`
- **Owner**: [Ch4va (Mohit Chava)](https://github.com/Ch4va)

## ✨ Features
- **Modern Dark Aesthetic**: Harmonious design palette featuring dark mode with vibrant orange/blue highlights (`--accent: #ff3300`, `--accent-2: #0066ff`).
- **Smooth GSAP Scrolling**: Uses GSAP and `ScrollTrigger` for advanced, performance-optimized animations and parallax effects.
- **Custom Cursor & Followers**: A magnetic custom cursor effect that interacts dynamically with links and hoverable items.
- **Floating Blur Shapes**: Interactive background blobs animated dynamically to bring the layout to life.
- **Grain Overlay**: Premium subtle noise overlay styling for visual depth.
- **Selected Works**: Responsive grids detailing highlight projects.
- **Responsive Layout**: Tailored viewports supporting mobile, tablet, and desktop displays.

## 🛠️ Technology Stack
- **Structure**: HTML5
- **Styling**: Modern CSS3 (Variables, Grid, Flexbox, Keyframes)
- **Interactions**: Vanilla JavaScript, GSAP (GreenSock Animation Platform)
- **Plugins**: GSAP ScrollTrigger, GSAP ScrollToPlugin
- **Typography**: Google Fonts (Space Grotesk & Syne)

## 💻 Running the Project
Simply open `index.html` in any web browser, or use a local dev server (e.g. VS Code Live Server, Python HTTP server, or standard `npm` tools):
```bash
# Using python:
python -m http.server 8000

# Or using Node.js:
npx serve .
```
